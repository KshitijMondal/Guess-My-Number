# Guess-My-Number
🕹️ Guess My Number! <br>
A retro-styled arcade game where you test your luck and logic against the computer. Built with clean vanilla JavaScript and a nostalgic 8-bit aesthetic. <br><br>

🎮 How to Play <br>
The Secret: The computer picks a secret number between 1 and 20. <br>
The Input: Type your guess into the box and hit the "Check!" button. <br><br>
The Clues:<br>
  "Too High!" 📈: Your guess is above the secret number.<br>
  "Too Low!" 📉: Your guess is below the secret number.<br>
  "Bingo!!!" 🎉: You found it! The screen turns green and reveals the number.<br>
The Stakes: You start with 20 points. Every wrong guess costs you 1 point.<br>
Restart: Use the "Again!" button to reset the game and try to beat your Highscore.<br><br>
🛠️ Technical Highlights<br>
JavaScript (ES6+): Uses Math.trunc and Math.random for precise number generation and event listeners for real-time interaction.<br>
CSS Styling: Features a responsive layout with a custom Google Font (Press Start 2P) for that authentic gaming look.<br>
DOM Manipulation: Dynamically updates the background color, element widths, and text content based on game state (Win/Loss).<br><br>
📁 File Structure<br>
index.html: The semantic structure of the game board.<br>
style.css: The "Retro-Dark" theme and layout logic.<br>
script.js: The engine behind the game logic, scoring, and highscore persistence.<br><br>
🚀 Setup<br>
Clone this repository.<br>
Open index.html in any modern web browser.<br>
Start guessing!<br>
